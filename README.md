# odkrywaj-anchors

A Claude Code skill that scaffolds 8 anchor markdown files plus optional hooks so Claude keeps full project memory across sessions.

The 8 anchors — STARTWORK, CONTEXT, FILEMAP, TECHSTACK, WORKFLOW, PROGRESS, DECISIONS, ENDWORK — live in your project root. STARTWORK is a router that tells Claude which of the others to read for the current task. Optional Claude Code hooks auto-load STARTWORK on every session start and trigger ENDWORK when you close out a session.
